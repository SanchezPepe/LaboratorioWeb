Contenedores HTML

Ejemplo gráfico de los diferentes tipos de contenedores en HTML

Body

Define el contenido principal del HTML, es la sección del que estará directamente visible en la página web.

Header

Header se utiliza para el encabezado, normalmente engloba al nav (Funciona como un div)

Nav

Nav se utiliza para la barra de navegación, es donde se ponen las distintas secciones de la página para que que el usuario pueda navegar

Main

En el main se pone el contenido principal, engloba todas las secciones principales de nuestra página

Section

Los seccions se usan para delimitar secciones generales dentro de nuestra página, engloba un tema en común o un grupo de elementos parecidos.

Article

Article se usa para delimitar un elemento específico dentro de un grupo de elementons parecidos (Como dentro de un seccion)

Aside

Aside se usa para crear una barra lateral que tenga contenido secundario

Div

Div es el contenedor más basico de todos, no indica lugar ni comportamiento, puede ser usado para cualquier cosa

Footer

Se utiliza para identificar la barra final de la página, generalmente se ponen terminos y condiciones o datos legales
